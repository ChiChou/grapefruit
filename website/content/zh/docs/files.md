# 文件浏览器与预览

## 文件浏览器

![文件浏览器](/files.webp)

文件浏览器提供目标应用文件系统的树形视图。在 iOS 上默认从应用的主目录开始。在 Android 上从根文件系统开始。

- 带面包屑导航的目录浏览
- 下载文件到本地
- 上传文件到设备
- 删除文件（非只读模式时）
- 点击任意文件用相应格式的查看器打开

## 支持的预览格式

### 十六进制视图

任意文件均可作为十六进制转储打开，显示地址列、十六进制字节和 ASCII 边栏，并支持跳转到任意偏移量。

### 文本编辑器

纯文本文件在 Monaco 编辑器中打开，带语法高亮。支持查看配置文件、脚本和日志文件。

### SQLite 编辑器

![SQLite 编辑器](/database.webp)

打开 `.db` 和 `.sqlite` 文件，带表浏览器。选择表可在数据网格中查看内容。针对数据库运行自定义 SQL 查询。

### Property List（plist）

将 Apple 属性列表文件（二进制和 XML）以格式化树形视图呈现，支持嵌套字典、数组和所有 plist 值类型。

### 图片预览

在浏览器中内联显示 PNG、JPEG、GIF、WebP 和其他常见图片格式。

### 音频预览

使用浏览器原生音频播放器播放音频文件（MP3、AAC、WAV 等）。

### 字体预览

渲染 TrueType（`.ttf`）和 OpenType（`.otf`）字体，以示例文本展示不同大小的字形。

### Assets.car（iOS）

![Assets.car](/assets.webp)

解析编译后的 asset catalog，以网格视图显示包含的图片、图标和其他资源。

### DEX 查看器

打开 `classes.dex` 文件，提供完整 DEX 分析视图——类浏览器、方法反汇编、字符串搜索、交叉引用和 AI 反编译。详见[分析与反编译](/docs/analysis)。

## APK 浏览器（Android）

以 zip 归档形式浏览应用 APK 文件的内容。提取单个条目（DEX 文件、原生库、资源、assets）用于进一步分析。点击 DEX 文件可直接在 DEX 查看器中打开。
